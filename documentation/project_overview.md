# Global Food Price Analysis Project (2010–2017)

## Overview
This project analyzes global food price trends across 55 countries between 2010 and 2017. The dataset contains over 740,000 records sourced from Kaggle and the World Bank.

The objective is to understand how food prices vary across countries, time, and commodities, and how exchange rates impact pricing behavior.

## Objectives
- Analyze global food price trends
- Compare commodity prices across countries
- Understand volatility in food markets
- Study the impact of exchange rates on pricing
- Build interactive Power BI dashboards

## Tools Used
- Power BI
- Microsoft Excel
- Power Query (M Language)
- DAX
- Kaggle Dataset
- World Bank API

## Outputs
- Market Overview Dashboard
- Price Analysis Dashboard
- Product Analysis Dashboard
